Mashruucan waa nidaam lagu maamulo shaqooyinka koorsada ee ardayda (Student Course Work), kaas oo ka kooban Backend (Node.js) iyo Frontend (Flutter).

## Features (Astaamaha Nidaamka)

### Admin (Maamulaha)
- **Users Management**: Ku darista, wax ka bedelka, iyo tirtirista isticmaaleyaasha (Admin/Student).
- **Courses Management**: Maamulista koorsooyinka.
- **Courseworks**: Soo gudbinta shaqooyinka ay tahay inay ardaydu qabtaan.
- **Grading**: Sixitaanka iyo siinta dhibcaha shaqooyinka la soo gudbiyay.

### Student (Ardayga)
- Dashboard muujinaya koorsooyinka iyo shaqooyinka u yaala.
- Soo gudbinta shaqooyinka (Submissions).
- Aragtida dhibcaha laga helay shaqadii hore la u gudbiyay.

## Tech Stack (Tiknoolajiyada la isticmaalay)
- **Backend**: Node.js, Express.js, MongoDB/PostgreSQL (API Layer).
- **Frontend**: Flutter (Mobile/Web/Desktop).
- **State Management**: Provider.
- **Theme**: Indigo & Teal theme.

## Setup Instructions (Sida loo rakibo)

### Backend
1. Tag folder-ka backend-ka.
2. Ku shub dependencies-ka: `npm install`.
3. Samey feylka `.env` oo geli xogta lagama maarmaanka ah (Database URL, Token secret).
4. Daar server-ka: `npm start`.

### Frontend (Flutter)
1. Tag folder-ka `flutter_frontend`.
2. Soo qaado packages-ka: `flutter pub get`.
3. Iska hubi in `lib/config/api_config.dart` uu ku yaalo URL-ka saxda ah ee API-ga.
4. Daar barnaamijka: `flutter run`.
